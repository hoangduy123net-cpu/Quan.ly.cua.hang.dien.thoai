# 📱 Quản Lý Cửa Hàng Điện Thoại

Hệ thống Quản lý Cửa hàng Điện thoại được xây dựng bằng **Visual Basic .NET (VB.NET)** trên **Visual Studio 2022** kết hợp với **SQL Server**. Phần mềm hỗ trợ quản lý các hoạt động cơ bản của một cửa hàng điện thoại như quản lý khách hàng, nhân viên, sản phẩm, hóa đơn và chi tiết hóa đơn.

## ✨ Chức năng chính
Đăng nhập hệ thống
- 👤 Quản lý khách hàng
- 📱 Quản lý điện thoại
- 🧾 Quản lý hóa đơn bán hàng
- 📄 Quản lý chi tiết hóa đơn
- 🔍 Tìm kiếm dữ liệu

  
  ## 🛠 Công nghệ sử dụng

- Visual Studio 2022
- Visual Basic .NET (WinForms)
- SQL Server
- Microsoft.Data.SqlClient
- ADO.NET

---

## 📋 Yêu cầu hệ thống

- Windows 10/11
- Visual Studio 2022
- .NET Framework (theo phiên bản của project)
- SQL Server 2019 hoặc SQL Server Express


- SQL Server Management Studio (khuyến nghị)
